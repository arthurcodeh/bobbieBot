# Dossier Members
Ce dossier contient les différents fichiers qui vont gérer les différents composants du robot. Chaque fichier correspond à un composant spécifique du robot, tel que les moteurs, les capteurs, ou encore les actionneurs.

---
## description de chaque fichier :
- `Members` : Ce fichier contient les déclarations des différentes classes et fonctions utilisées pour gérer les composants du robot. Il inclut également les bibliothèques nécessaires pour le bon fonctionnement du code.`
- `ServoSpecs` : Ce fichier contient les spécifications des servomoteurs utilisés dans le robot, telles que les angles de rotation, les vitesses, et les positions de repos mais aussi les limites de fonctionnement.
- `Tete`: Ce fichier contient la classe `Tete`, qui gère les mouvements de la tête du robot. Il inclut des fonctions pour faire tourner la tête, incliner la tête, et effectuer d'autres mouvements spécifiques à la tête du robot.
- `Bras`: Ce fichier contient la classe `Bras`, qui gère les mouvements des bras du robot. Il inclut des fonctions pour faire bouger les bras, saisir des objets, et effectuer d'autres mouvements spécifiques aux bras du robot.
